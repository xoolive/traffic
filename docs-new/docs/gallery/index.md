# Gallery

The Gallery is the story-first side of `traffic`: short, visual pages that explain
air traffic phenomena without assuming you are writing Python code.

We are rebuilding this section from the legacy Sphinx gallery and keeping the same
themes: trajectory patterns, ATC operations, traffic density, and Earth-system effects.

## Available now

- Content migration is in progress; most pages are currently placeholders.
- For a complete runnable narrative today, start with [Quickstart](/tutorials/quickstart/).
